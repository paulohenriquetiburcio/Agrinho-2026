# Plataforma Agrinho 2026 — Agro Forte, Futuro Sustentável

Esta é uma interface web responsiva desenvolvida para o **Concurso Agrinho 2026**. A plataforma funciona como um portal dinâmico para estudantes, conectando inovação tecnológica, educação e práticas agrícolas sustentáveis.

---

## 🚀 Sobre o Projeto

O objetivo principal da página é facilitar o acesso dos alunos às ferramentas tecnológicas do programa, promovendo o equilíbrio entre a alta produtividade no campo e a preservação do meio ambiente.

A interface conta com uma navegação fluida e uma **Área do Estudante** dividida em três pilares principais:
* **Redação Paraná:** Integração com Inteligência Artificial para envio e consulta de redações.
* **AgroRobótica:** Desafios tecnológicos voltados para a automação sustentável no campo.
* **Sustentabilidade:** Conteúdo educativo focado em práticas ecológicas e produtivas.

---

## 🛠️ Tecnologias Utilizadas

* **HTML5:** Estruturação semântica de toda a página (Tags como `<header>`, `<section>`, `<nav>`, `<footer>`).
* **CSS3 (Planejado/Vinculado):** Estilização visual, sistema de grids/flexbox para os cards e responsividade mobile através do arquivo `style.css`.
